# Визуализатор алгоритмов сортировки

Консольное приложение на C# для пошаговой визуализации и сравнения эффективности классических алгоритмов сортировки. Разработано в рамках учебной практики.

## Возможности

* **Пошаговая анимация:** Наглядная демонстрация процесса сортировки (сравнения выделяются красным цветом, перестановки — зеленым).
* **Режим «Гонки»:** Сравнительный бенчмарк четырех алгоритмов на едином массиве данных с формированием сводной таблицы результатов.
* **Реализованные алгоритмы:** Bubble Sort, Insertion Sort, Merge Sort, Quick Sort.

## Стек технологий

* **Язык и платформа:** C# 14, .NET 10.0
* **Тестирование:** xUnit
* **Развертывание:** Docker

## Сборка и запуск (Docker)

Проект полностью контейнеризирован. Для запуска не требуется установка .NET SDK, достаточно наличия Docker.

1. Склонируйте репозиторий и перейдите в директорию проекта:
```bash
git clone https://github.com/DOPIAS/SortingVisualizer.git
cd SortingVisualizer
```

2. Выполните сборку образа и запустите контейнер:
```bash
docker build -t sort-app -f SortingVisualizer/Dockerfile .
docker run -it sort-app
```

## Тестирование

В проекте реализовано автоматизированное тестирование алгоритмического ядра (включая обработку граничных условий) и интеграционное тестирование взаимодействия модулей. 

Для запуска комплекса тестов выполните в корневой директории проекта команду:
```bash
dotnet test
```
